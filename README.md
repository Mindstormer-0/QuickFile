# QuickFile FTP Server & Client With GUI



### Compliling & Running

- **Windows:**
- Compile: ```javac -cp "lib\*" src\*.java```
- Run (client): ```java -cp "src;lib\*" Driver```
- Run (server): ```java -cp "src;lib\*" Server```

- **OSX/Linux:**
- Compile: ```javac -cp "lib/*" src/*.java```
- Run (client): ```java -cp "src:lib/*" Driver```
- Run (server): ```java -cp "src:lib/*" Server```
